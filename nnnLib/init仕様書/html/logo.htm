init - logo


共通設定
その他設定 なし それぞれのとこを参照


専用設定
名前 サブ デフォルト値 解説
modename logo モード名、変更不可。省略不可
nextMode title ロゴ表示後のモードを番号または名前で設定する
defaultUseBGColor 0 フェードイン、フェードアウト時のBGカラーのデフォルトの指定方法を設定する 0:指定色 1:画像左上の色
defaultFadeInTime 10 フェードインのデフォルトの長さをフレーム数で設定する
defaultFadeOutTime 10 フェードアウトのデフォルトの長さをフレーム数で設定する
defaultKeepTime 20 表示時間のデフォルトの長さをフレーム数で設定する
defaultFadeInColor[RGB] 255 フェードイン時のデフォルトの色を設定する
defaultFadeOutColor[RGB] 255 フェードアウト時のデフォルトの色を設定する
useCustomBGColorFlag 0 1:フェードイン、フェードアウト時の背景色の表示方法を絵ごとに設定する
fadeInBGColorFlag[n] defaultUseBGColor n番目の絵の背景表示方法を設定する。defaultUseBGColorと同じ動作
fadeOutBGColorFlag[n] defaultUseBGColor
useCustomFileFlag 0 1:絵のファイル名を個別に指定する。0:logo1,logo2...
filename[n] logo[n] n番目の絵のファイル名を設定する
useCustomSkipFlag 0 1:絵ごとにスキップ処理の仕方を設定する
skipOk[n] 1 n番目の絵のスキップ処理を 0:禁止する 1:許可する 2〜その回数目の起動から許可する
useCustomTimeFlag 0 1:絵ごとに表示時間を設定する
fadeInTime[n] defaultFadeInTime n番目の絵のフェードイン時間をフレーム数で設定する
fadeOutTime[n] defaultFadeOutTime n番目の絵のフェードアウト時間をフレーム数で設定する
keepTime[n] defaultKeepTime n番目の絵の表示時間をフレーム数で設定する
useCustomColorFlag 0 1:絵ごとにフェードイン、フェードアウト時の背景色を設定する
fadeInColor[RGB][n] defaultFadeInColor[RGB] n番目の絵のフェードイン時の背景色を設定する
fadeOutColor[RGB][n] defaultFadeOutColor[RGB] n番目の絵のフェードアウト時の背景色を設定する
