init - button


[ボタン名] それぞれのボタン名に変換
[n] 1:通常時 2:マウスが乗った時 3:決定(クリック)時 4:選択不可時
[k] 1-3


名前 サブ デフォルト値 解説
[ボタン名]Filename ボタンのグラフィックファイル名(システム dwqファイル)を拡張子なしで設定
[ボタン名]ClickTime 8 クリックしてからボタンが押された動作を完了するまでの時間をフレーム数で設定
[ボタン名]ExitWait 0 ボタンからカーソルがはずれたと判断されるまでの待ち時間を設定
[ボタン名]SizeX 16 ボタンのサイズを設定
[ボタン名]SizeY 16
[ボタン名]PrintX 0 ボタンの表示位置を設定
[ボタン名]PrintY 0
[ボタン名]CheckDeltaX 0 ボタンの反応範囲開始位置を相対位置で指定
[ボタン名]CheckDeltaY 0
[ボタン名]CheckPicX 0 ボタンの反応マスク画像の位置を指定
[ボタン名]CheckPicY 0
[ボタン名]CheckSizeX [ボタン名]SizeX ボタンの反応マスクサイズを指定
[ボタン名]CheckSizeY [ボタン名]SizeY
[ボタン名]PrintMode 2 表示モード(現在2のみ使用可)
[ボタン名]IsCancelButton 0 1:このボタンをキャンセルボタンに設定(画面の任意の位置での右クリックに反応する)
[ボタン名]IsUpButton 0 1:このボタンを上ボタンに設定(画面の任意の位置でのマウスホイールに反応する)
[ボタン名]IsDownButton 0 1:このボタンを下ボタンに設定(画面の任意の位置でのマウスホイールに反応する)
[ボタン名]UpDownReverse 0 1:上下ボタンの動作を反転させる
[ボタン名]TateYoko 0 1:グラフィックの並びを縦横いれかえる(主に縦長なボタンの場合に使う)
[ボタン名]PicSrcX[n] sizeX*(n-1) 画像データの開始位置を設定
[ボタン名]PicSrcY[n] 0
[ボタン名]AnimeType[n] 1 ボタンのアニメタイプ1-8で設定(アニメタイプは別表を参照)
[ボタン名][ボタン名]AnimePattern[n] 1 ボタンアニメのグラフィックパターン数を設定
[ボタン名]AnimeSpeed[n] 1 ボタンアニメの1パターンの表示時間をフレームで設定
[ボタン名]AccelKey[k] 0 ショートカットキーをキー番号または、1文字の数字、アルファベット大文字で設定(最大3つまで設定可能)
[ボタン名]ClickSound 0 クリックした瞬間に出す音を設定(systemSe.txtに対応)
[ボタン名]CannotClickSound 0 選択不可ボタンを押したときに出す音を設定(systemSe.txtに対応)
[ボタン名]EnterSound 0 ボタンにカーソルがはいったと判断されたときに出す音を設定(systemSe.txtに対応)
[ボタン名]ExitSound 0 ボタンからカーソルがはずれたと判断されたときに出す音を設定(systemSe.txtに対応)
[ボタン名]EnterWait 0 ボタンにカーソルがはいったと判断されるまでの待ち時間を設定
[ボタン名]ExitWait 0 ボタンからカーソルがはずれたと判断されるまでの待ち時間を設定


アニメタイプ 繰り返し 折り返し オーバーラップ
1 有 通常 無
2 1回のみ 通常 無
3 有 折り返し 無
4 1回のみ 折り返し 無
5 有 通常 オーバーラップ
6 1回のみ 通常 オーバーラップ
7 有 折り返し オーバーラップ
8 1回のみ 折り返し オーバーラップ


accelKey設定での注意
数値または、キーの文字で指定する。
アルファベットは大文字。(大文字=全角ではありません)

このとき、「0」のキーが指定できないので
「0」を指定したいときは、「0」の文字コード48で指定します

また、「,」(カンマ)や「 」(スペース)もinitに使用できないので
数値で指定します
