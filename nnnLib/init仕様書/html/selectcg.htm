init - selectcg


共通設定
上下ボタン設定 必要 それぞれのとこを参照あとでリンクをつける
戻るボタン設定 必要 それぞれのとこを参照
背景スクリプト設定 可能 それぞれのとこを参照
BGM設定 可能 それぞれのとこを参照
画面切り替え時フェード設定 可能 それぞれのとこを参照
画面切り替え時音声設定 可能 それぞれのとこを参照
その他設定 なし それぞれのとこを参照


専用設定
名前 サブ デフォルト値 解説
modename selectcg モード名、変更不可。省略不可
cgPianoSoundFlag 0 1:選択可能サムネイルにマウスカーソルが乗ったときにシステム標準のピアノ音を出す
filenameBG cg_bg 背景画像ファイル名(背景スクリプトを使用しない場合に有効)
blockNumberX 4 表示するサムネイルの個数を横、縦で指定する
blockNumberY 3
picMustPrintPercent 0 手に入れていない絵であっても、サムネイルを表示する場合には、ここに透明度を1-100の値で設定する
addPicFlag 0 画面に追加の絵を表示する場合に1を設定する。選択されたCGキャラごとに違う絵（タイトルや名前など）を表示する場合に使う
addPicPrintPercent 0 表示する絵の透明度を%で指定。ただし0の場合にはマスク処理なし
addPicPrintX 217 追加絵の表示位置を指定
addPicPrintY 26
addPicSizeX 355 追加絵のキャラ1人分のサイズを指定
addPicSizeY 22
addPicSrcNumX 1 追加絵でキャラ絵が縦横いくつずつ並んでいるかを指定
addPicSrcNumY CGキャラの数
filenameAdd cgbg_title 追加絵のファイル名
printX 85 1つめのサムネイルエリアの表示位置を指定
printY 113
nextX 166 次のサムネイルエリアまでの距離を指定
nextY 124
sizeX 133 サムネイル画像のサイズを指定
sizeY 100
picPrintX 0 サムネイルエリアでのサムネイルの表示位置を相対座標で指定
picPrintY 0
picSizeX sizeX 未実装
picSizeY sizeY
cursorPrintX picPrintX サムネイルエリアでのカーソル表示位置を相対座標で指定
cursorPrintY picPrintY
cursorSizeX picSizeX(sizeX) カーソルサイズを指定
cursorSizeY picSizeY(sizeY)
futaPrintFlag 1 取得していないCG部分をふたで隠す場合に1を設定する、ふたの表示位置は、サムネイル表示エリア。マスク処理しない場合には2を指定する
filenameFuta ta_futa1 フタのファイル名
selectPrintMode 1 カーソルの表示モードを指定 0:なし 1:色 2:絵
cursorColorR 230 カーソルの表示モードが1の場合に、カーソルの色をRGBで指定
cursorColorG 0
cursorColorB 50
fileNameCursor ta_cg_cur カーソル表示モードが2の場合にカーソルファイル名を指定
cursorPrintType 0 カーソル表示モードが2の場合にカーソルの表示方法を0-2で指定 0:通常 1:半透明(selectPercentで指定) 2:加算
selectPercent 50 カーソル表示モードが2ので、カーソルの表示方法が1の場合にカーソルの透明度を指定
totalPercentPrintFlag 1 CG回収%を表示する場合1を設定する。しない場合は0
fileNameTotalPercent ta_font_cg1 CG回収%表示用の、フォントグラフィックファイル名
totalPercentFontSizeX 16 フォントサイズ
totalPercentFontSizeY 22
totalPercentFontNextX totalPercentFontSizeX 次の文字までのサイズ
totalPercentPrintX 688 表示位置
totalPercentPrintY 25
differencePrintFlag 1 差分情報表示フラグ 0:表示しない 1:差分があるものにのみ表示 2:全てに表示
differencePrintByGetFlag 0 1枚でも取得したものだけ差分情報を表示する場合には、1を指定する
differenceFontSizeX 21 差分情報用フォントサイズ
differenceFontSizeY 27
differenceFontNextX differenceFontSizeX 次の文字までのサイズ
differencePrintX 0 サムネイルエリアでの差分情報表示座標を相対座標で指定する
differencePrintY 0
fileNameDifferenceFont ta_sl_font3 差分情報用フォントグラフィックファイル名
pagePrintFlag 0 ページ数を表示する場合は1、表示しない場合は0
pageFontSizeX 21 ページ数表示用フォントサイズ
pageFontSizeY 27
pageFontNextX pageFontSizeX 次の文字までのサイズ
pagePrintX 0 ページ数表示座標
pagePrintY 0
fileNamePageFont ta_sl_font3 ページ表示用フォントグラフィックファイル名
pageMaxPrintFlag 0 最大ページ数を表示する場合には1、表示しない場合には0を設定
pageMaxPrintX pagePrintXから3文字分右 最大ページ数表示座標
pageMaxPrintY pagePrintY
slashPrintFlag 0 スラッシュを表示する場合には1を設定する
slashPrintX pagePrintXから2文字分右 スラッシュを表示する座標
slashPrintY pagePrintY
miniPicTransFlag 0 サムネイルをマスク付き画像として処理する場合には1を設定する
