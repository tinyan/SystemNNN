init - cursor


[n] カーソル種類
1 通常おくりカーソル
2 通常自動カーソル
3 全画面送りカーソル
4 全画面自動カーソル
名前 サブ デフォルト値 解説
name messageCursor 省略しないこと
filenameCursor[n] mes_cur カーソルグラフィックファイル名を設定
printX[n] 0 送りカーソルの表示位置を設定
printY[n] 0
sizeX[n] 32 送りカーソルのサイズを設定
sizeY[n] 32
fullFixX 0 全画面の時、0:メッセージからの相対位置 1:そのままの座標
fullFixY 0
animePattern[n] 1 送りカーソルのアニメパターン数
animeSpeed[n] 1 送りカーソルのアニメスピードをフレームで設定
animeSpeed[n] 1 1:送りカーソルのアニメが折り返す
percentStart[n] 100 送りカーソルの開始透明度
percentEnd[n] 100 送りカーソルの終了透明度
percentLoop[n] 1 1:送りカーソルの透明度計算処理がループさせる
percentReverse[n] 0 1:送りカーソルの透明度計算処理が折り返し
