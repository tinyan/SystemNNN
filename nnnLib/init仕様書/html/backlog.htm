init - backlog


共通設定
背景スクリプト設定 可能 それぞれのとこを参照
BGM設定 可能 それぞれのとこを参照
その他設定 なし それぞれのとこを参照


専用設定
名前 サブ デフォルト値 解説
modename backlog モード名、変更不可。省略不可
returnSoundNumber 0 バックログから戻るときに音を出す場合にはシステムサウンドの番号を設定する
bgMode 0 背景処理 0:塗りつぶし 1:絵 2:キャプチャー画像
背景処理 塗りつぶしモード (bgMode=0)
backColorR 0 色をRGBで設定
backColorG 0
backColorB 0
背景処理 絵モード (bgMode=1)
backlogBGFileName bg_backlog 背景画像ファイル名
背景処理 キャプチャーモード (bgMode=2)
backScreenEffectNega 0 1:キャプチャー画像にネガ処理をかける 0:処理しない
backScreenEffectGrey 0 1:キャプチャー画像に白黒化処理をかける 0:処理しない
backMultiR 100 キャプチャー画像にかける乗算値を0 〜 100で設定する
backMultiG 100
backMultiB 100
backDeltaR 0 キャプチャー画像にかける加減算値を -255 〜 255で設定する
backDeltaG 0
backDeltaB 0
fontSize 24 フォントサイズを設定
printLines 15 表示行数を設定
printX 110 ログ表示開始座標
printY 32
nextX 0 次の行までの距離
nextY fontSize
titlePrintFlag 1 1:タイトルメッセージを表示する 0:表示しない
titlePrintX 画面中央から3文字左 タイトルメッセージ表示座標
titlePrintY printY
titleMessage ※1 タイトル文章を３１文字(62byte)までで設定する
titlePicPrintFlag 0 1:バックログタイトル画像を表示する
titlePicFileName ta_backlog_title バックログタイトル画像ファイル名を設定
titlePicPrintX 0 バックタイトル画像表示座標
titlePicPrintY 0
titlePicUpperFlag 0 1:バックタイトル画像を文字の上に表示する
barMessage ※2 セパレーターメッセージを３１文字(62byte)までで設定する
firstMessageNumber 0 開始メッセージを指定するときに1を設定する。開始メッセージが62byteを超える場合には複数に分割し、その分割個数を指定する
firstMessage[n] なし n番目の分割開始メッセージを指定する。分割開始メッセージは31文字(62byte)までで、かつ合計510byteまで(表示はみ出さないように注意)[デフォルト:※3]
upArrowPrintX 画面中央 上カーソル表示座標
upArrowPrintY printYから2行下
downArrowPrintX 画面中央 下カーソル表示座標
downArrowPrintY printYから(表示行数+3)行下
updownArrowSizeX fontSize 上下カーソルサイズ
upDownArrowSizeY fontSize
upArrowMessage #水▲ 上カーソルの文字を設定します。
downArrowMessage #水▼ 下カーソルの文字を設定します。
updownArrowPicFlag 0 1:上下カーソルをグラフィックで表示します 0:文字のまま
filenameUpDownArrow ta_backlog_updown 上下カーソルファイル名を設定します
updownArrowAnimeFlag 0 1:上下カーソルをアニメさせます
upArrowFileName ta_backlog_up 上下カーソルをアニメさせる場合には個別にファイルを設定します(上カーソル)
downArrowFileName ta_backlog_down (下カーソル)
updownArrowAnimePattern 2 上下カーソルアニメのグラフィックパターン数
updownArrowAnimeType 5 上下カーソルアニメのアニメ種類(ボタンと同じ設定)
updownArrowAnimeSpeed 10 上下カーソルアニメのアニメフレーム数
voiceSizeX fontSize 音声ボタンサイズを設定
voiceSizeY fontSize
voiceMessage ♪ 文字音声ボタンの文字を設定します。
voiceFlag 1 1:文字音声ボタンを押したときに音声がでます 0:音声なし
voicePrintX -fontX 音声ボタンの表示位置をメッセージからの相対座標で指定
voicePrintY 0
voicePicFlag 0 1:音声ボタンをグラフィックで表示します 0:文字のまま
filenameVoice ta_backlog_updown 音声ボタンのファイル名を設定
voicePicAnimeFlag 0 1:音声ボタンをアニメさせます
voicePicAnimePattern 2 音声ボタンのアニメのグラフィックパターン数
voicePicAnimeType 5 音声ボタンのアニメのアニメ種類(ボタンと同じ設定)
voicePicAnimeSpeed 10 音声ボタンアニメのアニメフレーム数
